(WIP) Archive of solutions to Project Euler problems in R. 

Code has been optimized so that each problem should not take more than 10 seconds to be solved. 
That said, algorithms may not be 100% optimal; I have used inefficient constructions like list comprehensions where it does not seem to make much of a difference to the runtime. 

Some problems may print the solutions as individual digits in a list rather than a simple number. When copy and pasting from the R console, the spaces between the digits need to be deleted before the Project Euler website will accept them as solutions.
